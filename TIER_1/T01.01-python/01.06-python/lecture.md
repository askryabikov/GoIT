Создание виртуального пространства в VSCode на Windows:
0. В новой папке создать файл main.py и в консоли прописать: python3 -m venv .venv
1. source .venv/Activate.ps1 - активировать виртуальное пространство через терминал


Загрузка модулей через терминал в VScode
1. pip3 freeze > requirements.txt - выдаст список загуженных модулей в текстовом файле
2. pip3 intall -r requirements.txt - загрузит все модули из этого спика
3. pip3 list - проверить загруженные модули
4. pip3 install django - загрузить библиотеку Django

В самом файле прописать код на загрузку необзодимого модуля, например:
