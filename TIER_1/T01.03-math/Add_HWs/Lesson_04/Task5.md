Задача 5. Упрощение логического выражения

Дано:
Упростить выражение:
F = (¬p ∨ q) ∧ (p ∨ q)

Нужно:
Получить самую простую форму.

------------------------------------------------------------

Упрощение логического выражения

Шаг 1. Применяем дистрибутивность

Раскроем скобки:
F = ((¬p ∨ q) ∧ p) ∨ ((¬p ∨ q) ∧ q)

Шаг 2. Упростим каждую часть отдельно

Часть 1:
(¬p ∨ q) ∧ p

Распределим p:
(¬p ∧ p) ∨ (q ∧ p)

¬p ∧ p = 0 (противоречие)
q ∧ p = p ∧ q

Значит:
(¬p ∨ q) ∧ p = p ∧ q

Часть 2:
(¬p ∨ q) ∧ q

Распределим q:
(¬p ∧ q) ∨ (q ∧ q)

q ∧ q = q

Значит:
(¬p ∧ q) ∨ q

По поглощению (A ∨ (A ∧ B) = A):
(¬p ∧ q) ∨ q = q

Шаг 3. Объединяем две части

F = (p ∧ q) ∨ q

Снова применяем поглощение:
(p ∧ q) ∨ q = q

------------------------------------------------------------

Ответ

(¬p ∨ q) ∧ (p ∨ q) ≡ q

Финальная форма: q
