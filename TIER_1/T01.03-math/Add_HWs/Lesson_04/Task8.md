```md
Задача 8. Рассадка

Дано:
Есть 6 студентов:
S = {s1, s2, s3, s4, s5, s6}

Есть 3 стула в ряду:
(1), (2), (3)

Нужно:
Посадить 3 разных студентов на 3 разные места. Порядок важен.
Сколько существует вариантов рассадки?

------------------------------------------------------------

Решение

Шаг 1. Определяем тип задачи

Выбираем k элементов из n, и порядок важен → это размещения.

Формула:
A(n, k) = n! / (n - k)!

Шаг 2. Подставляем значения

n = 6
k = 3

A(6, 3) = 6! / (6 - 3)!
        = 6! / 3!

Шаг 3. Считаем факториалы

6! = 720
3! = 6

Шаг 4. Делим

A(6, 3) = 720 / 6 = 120

Ответ

Существует 120 разных способов посадить трёх студентов на три места.
```
