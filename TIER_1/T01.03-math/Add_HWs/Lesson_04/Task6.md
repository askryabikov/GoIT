```md
Задача 6. Преобразование импликации

Дано:
Преобразовать выражение:
p → (q → r)
в форму без импликаций.

Вопрос:
Эквивалентны ли формулы:
¬(p ∧ q)
и
¬p ∨ ¬q ?

------------------------------------------------------------

Преобразование импликации

Шаг 1. Используем определение импликации

Импликация:
a → b ≡ ¬a ∨ b

Шаг 2. Преобразуем внутреннюю импликацию

q → r = ¬q ∨ r

Шаг 3. Подставляем результат во внешнюю импликацию

p → (q → r) = p → (¬q ∨ r)

Шаг 4. Снова применяем определение импликации

p → X = ¬p ∨ X

Значит:
p → (¬q ∨ r) = ¬p ∨ (¬q ∨ r)

Шаг 5. Убираем лишние скобки

По ассоциативности ∨:
¬p ∨ (¬q ∨ r) ≡ ¬p ∨ ¬q ∨ r

------------------------------------------------------------

Ответ

p → (q → r) ≡ ¬p ∨ ¬q ∨ r

------------------------------------------------------------

Ответ на вопрос (законы де Моргана)

Да, эквивалентны:
¬(p ∧ q) ≡ ¬p ∨ ¬q

Это первый закон де Моргана.
```
