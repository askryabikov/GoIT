```md
Задача 10. Степени вершин, смежность, инцидентность

Дано:
Неориентированный граф.

Множество вершин:
V = {A, B, C, D}

Множество рёбер:
E = {(A,B), (A,C), (B,C), (B,D), (C,D)}

Нужно:
1) Для каждой вершины перечислить всех соседей (смежные вершины).
2) Найти степень каждой вершины: deg(A), deg(B), deg(C), deg(D).
3) Проверить теорему о сумме степеней:
   sum_{v in V} deg(v) = 2 * |E|

------------------------------------------------------------

Решение

Шаг 1. Определения

Граф: G = (V, E)

Вершины V — объекты (города).
Рёбра E — связи между вершинами (дороги).

Две вершины называются смежными, если между ними есть ребро.

Степень вершины deg(v) — количество рёбер, инцидентных вершине
(сколько рёбер к ней “подходит”).

В неориентированном графе каждое ребро (u, v) добавляет:
+1 к deg(u) и +1 к deg(v).

Шаг 2. Выписываем рёбра

(A, B)
(A, C)
(B, C)
(B, D)
(C, D)

Шаг 3. Соседи (смежные вершины) для каждой вершины

Для A:
Рёбра с A: (A,B), (A,C)
N(A) = {B, C}

Для B:
Рёбра с B: (A,B), (B,C), (B,D)
N(B) = {A, C, D}

Для C:
Рёбра с C: (A,C), (B,C), (C,D)
N(C) = {A, B, D}

Для D:
Рёбра с D: (B,D), (C,D)
N(D) = {B, C}

Шаг 4. Степени вершин

deg(A) = |N(A)| = 2
deg(B) = |N(B)| = 3
deg(C) = |N(C)| = 3
deg(D) = |N(D)| = 2

Шаг 5. Проверяем теорему о сумме степеней

Сумма степеней:
deg(A) + deg(B) + deg(C) + deg(D) = 2 + 3 + 3 + 2 = 10

Количество рёбер:
|E| = 5
2|E| = 2 * 5 = 10

Сравнение:
sum deg(v) = 10
2|E| = 10

Равенство выполняется.

------------------------------------------------------------

Ответы

Соседи:
N(A) = {B, C}
N(B) = {A, C, D}
N(C) = {A, B, D}
N(D) = {B, C}

Степени:
deg(A) = 2
deg(B) = 3
deg(C) = 3
deg(D) = 2

Проверка теоремы:
sum deg(v) = 10 = 2|E|
```
