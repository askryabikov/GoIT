```md
Задача 7. Выбор команды

Дано:
Есть 5 студентов:
S = {s1, s2, s3, s4, s5}

Нужно:
Сформировать команду из 2 человек. Порядок не важен.
Сколько разных команд можно составить?

------------------------------------------------------------

Решение

Шаг 1. Определяем тип задачи

Выбираем k элементов из n без учёта порядка → это сочетания.

Формула:
C(n, k) = n! / (k! * (n - k)!)

Шаг 2. Подставляем значения

n = 5
k = 2

C(5, 2) = 5! / (2! * 3!)

Шаг 3. Считаем факториалы

5! = 120
2! = 2
3! = 6

Шаг 4. Считаем значение

C(5, 2) = 120 / (2 * 6) = 120 / 12 = 10

Ответ

Можно составить 10 разных команд.
```
