```md
Задача 3

Дано:
Система:
  x + y + z = 6
  2x - y + 3z = 14
  3x + 4y - z = 2

Нужно:
1) Найти решение системы методом Гаусса.
2) Ответить: как выглядит приведённая ступенчатая форма этой системы?

------------------------------------------------------------

Решение (метод Гаусса)

Шаг 1. Записываем расширенную матрицу системы:

[ 1   1   1 |   6 ]
[ 2  -1   3 |  14 ]
[ 3   4  -1 |   2 ]

Шаг 2. Обнуляем элементы под первым пивотом (1-й столбец).

Операции:
R2 := R2 - 2R1
R3 := R3 - 3R1

Получаем:

[ 1   1   1 |   6 ]
[ 0  -3   1 |   2 ]
[ 0   1  -4 | -16 ]

Шаг 3. Делаем пивот во 2-м столбце и обнуляем ниже.

Поменяем местами R2 и R3 (не обязательно, но удобнее):
R2 ↔ R3

[ 1   1   1 |   6 ]
[ 0   1  -4 | -16 ]
[ 0  -3   1 |   2 ]

Обнуляем элемент -3 под 1:
R3 := R3 + 3R2

Получаем:

[ 1   1    1 |   6 ]
[ 0   1   -4 | -16 ]
[ 0   0  -11 | -46 ]

Шаг 4. Обратный ход (находим z, y, x).

Из третьей строки:
-11z = -46  ->  z = 46/11

Из второй строки:
y - 4z = -16
y = -16 + 4z
y = -16 + 4*(46/11) = (-176 + 184)/11 = 8/11

Из первой строки:
x + y + z = 6
x = 6 - y - z

Шаг 5. Подставляем и считаем до конца.

z = 46/11
y = 8/11
x = 12/11

Ответ

x = 12/11
y = 8/11
z = 46/11

Приведённая ступенчатая форма (один из вариантов):

[ 1   1    1 |   6 ]
[ 0   1   -4 | -16 ]
[ 0   0  -11 | -46 ]
```
