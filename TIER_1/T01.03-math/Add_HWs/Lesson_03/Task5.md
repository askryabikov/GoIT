Задача 5

Дано:
Точки:
(1, 2), (2, 4), (3, 5)

Нужно:
Найти коэффициенты прямой y = kx + b методом наименьших квадратов (МНК).

Вопрос:
Почему в МНК минимизируют сумму квадратов ошибок, а не просто сумму модулей?

------------------------------------------------------------

Ответы

Шаг 1. Составляем таблицу значений и считаем суммы.

Имеем три точки:
x1 = 1, y1 = 2
x2 = 2, y2 = 4
x3 = 3, y3 = 5

Считаем:
sum(x_i)   = 1 + 2 + 3 = 6
sum(y_i)   = 2 + 4 + 5 = 11
sum(x_i^2) = 1^2 + 2^2 + 3^2 = 1 + 4 + 9 = 14
sum(x_i y_i) = 1·2 + 2·4 + 3·5 = 2 + 8 + 15 = 25

Также n = 3 (количество точек).

Шаг 2. Записываем нормальные уравнения для y = kx + b.

Стандартная форма:
k·sum(x_i^2) + b·sum(x_i) = sum(x_i y_i)
k·sum(x_i)   + b·n        = sum(y_i)

Подставляем числа:
14k + 6b = 25
6k  + 3b = 11

Шаг 3. Решаем систему для k и b.

Из второго уравнения:
6k + 3b = 11
Делим на 3:
2k + b = 11/3
b = 11/3 - 2k

Подставляем b в первое уравнение:
14k + 6(11/3 - 2k) = 25

Раскрываем скобки:
14k + 22 - 12k = 25
2k + 22 = 25
2k = 3
k = 3/2

Находим b:
b = 11/3 - 2·(3/2)
b = 11/3 - 3
b = 11/3 - 9/3 = 2/3

Шаг 4. Записываем уравнение прямой.

y = (3/2)x + 2/3

------------------------------------------------------------

Ответ на вопрос

МНК минимизирует сумму квадратов ошибок:
sum( y_i - (k x_i + b) )^2

Квадрат используют потому что:
- функция ошибки становится гладкой (дифференцируемой) по k и b;
- большие отклонения «штрафуются» сильнее;
- получается система линейных уравнений (нормальные уравнения), которую можно решить аналитически.
