Задача 6

Дано:
A = [[1, 1],
     [1, 2],
     [1, 3]]

b = [2, 2, 4]

Вопрос:
Что означает проекция вектора b на Col(A)?

------------------------------------------------------------

Решение

Шаг 1. Записываем A и b в матричном виде.

A = [[1, 1],
     [1, 2],
     [1, 3]]

b = [2, 2, 4]

Ищем вектор параметров x = [b0, k]^T такой, что Ax ≈ b в смысле МНК.

Шаг 2. Записываем нормальные уравнения.

Метод наименьших квадратов даёт систему:
(A^T A) x = A^T b

Шаг 3. Находим A^T и A^T A.

A^T = [[1, 1, 1],
       [1, 2, 3]]

Теперь умножаем:
A^T A = [[1, 1, 1],
         [1, 2, 3]]  *  [[1, 1],
                         [1, 2],
                         [1, 3]]
      = [[3,  6],
         [6, 14]]

Шаг 4. Находим A^T b.

A^T b = [[1, 1, 1],
         [1, 2, 3]] * [2, 2, 4]
      = [2 + 2 + 4,
         1·2 + 2·2 + 3·4]
      = [8,
         2 + 4 + 12]
      = [8, 18]

Шаг 5. Записываем нормальные уравнения как систему 2×2.

(A^T A) x = A^T b

[[3,  6],      [b0]     [ 8]
 [6, 14]]  *   [ k]  =  [18]

То есть:
3b0 + 6k  = 8
6b0 + 14k = 18

Шаг 6. Решаем систему для b0 и k.

Вычтем 2-кратное первое уравнение из второго:
(6b0 + 14k) - 2(3b0 + 6k) = 18 - 2·8
6b0 + 14k - 6b0 - 12k = 18 - 16
2k = 2
k = 1

Подставляем k = 1 в первое уравнение:
3b0 + 6·1 = 8
3b0 = 2
b0 = 2/3

Итого:
x_hat = [b0, k]^T = [2/3, 1]^T

Ответ

МНК-оценка параметров:
b0 = 2/3, k = 1

Прямая:
y = x + 2/3

Проекция b на Col(A) — это вектор A x_hat:
он лежит в столбцовом пространстве A и является ближайшим к b
по обычной евклидовой дистанции (минимизирует ||b - Ax||_2).
```
