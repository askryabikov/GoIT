```md
Задача 4

Дано:
Система:
  x + 2y = 7
  3x - y = 8

Нужно:
1) Найти решение системы с помощью обратной матрицы.
2) Ответить: можно ли найти решение через A^(-1)? Если да — выписать A^(-1) явно.

------------------------------------------------------------

Решение

Шаг 1. Записываем систему в виде Ax = b

A = [[ 1,  2],
     [ 3, -1]]

x = [[x],
     [y]]

b = [[7],
     [8]]

Шаг 2. Находим det(A) (для 2×2)

det(A) = 1·(-1) - 2·3
       = -1 - 6
       = -7

Так как det(A) ≠ 0, матрица A обратима.

Шаг 3. Находим A^(-1)

Если A = [[a, b],
          [c, d]],
то A^(-1) = (1/det(A)) * [[ d, -b],
                          [-c,  a]]

Для нашей матрицы:
A^(-1) = (1/(-7)) * [[-1, -2],
                     [-3,  1]]
      = (-1/7) * [[-1, -2],
                  [-3,  1]]

Шаг 4. Находим решение x = A^(-1) b

Сначала умножаем внутреннюю матрицу на b:

[[-1, -2],   [[7],   [[(-1)·7 + (-2)·8],
 [-3,  1]] *  [8]] =  [(-3)·7 + 1·8]]

= [[-7 - 16],
   [-21 + 8]]
= [[-23],
   [-13]]

Теперь умножаем на скаляр -1/7:

x = (-1/7)·(-23) = 23/7
y = (-1/7)·(-13) = 13/7

Ответ

Да, так как det(A) ≠ 0, матрица A обратима.

A^(-1) = [[ 1/7,  2/7],
          [ 3/7, -1/7]]

Решение системы:
x = 23/7
y = 13/7
```
