```md
Задача 7

Дано:
Точки:
(1, 3), (2, 5), (4, 6)

Нужно:
Найти прямую y = kx + b методом наименьших квадратов (МНК).

Вопрос:
Как интерпретировать «минимизировать ошибку» в терминах расстояния в R^n?

------------------------------------------------------------

Решение

Шаг 1. Считаем нужные суммы.

Точки:
(1, 3)
(2, 5)
(4, 6)

Суммы:
Sx  = sum(x_i)     = 1 + 2 + 4 = 7
Sy  = sum(y_i)     = 3 + 5 + 6 = 14
Sxx = sum(x_i^2)   = 1^2 + 2^2 + 4^2 = 1 + 4 + 16 = 21
Sxy = sum(x_i y_i) = 1·3 + 2·5 + 4·6 = 3 + 10 + 24 = 37
n = 3

Шаг 2. Записываем нормальные уравнения для y = kx + b.

k·Sxx + b·Sx = Sxy
k·Sx  + b·n  = Sy

Подставляем числа:
21k + 7b = 37
7k  + 3b = 14

Шаг 3. Решаем систему для k и b.

Из второго уравнения:
7k + 3b = 14
3b = 14 - 7k
b = (14 - 7k)/3

Подставляем в первое:
21k + 7*(14 - 7k)/3 = 37

Приводим к общему знаменателю 3:
(63k + 98 - 49k)/3 = 37
(14k + 98)/3 = 37

Умножаем на 3:
14k + 98 = 111
14k = 13
k = 13/14

Находим b:
b = (14 - 7k)/3
b = (14 - 7*(13/14))/3

7*(13/14) = 13/2, значит:
b = (14 - 13/2)/3
14 = 28/2
b = ((28/2 - 13/2)/3) = (15/2)/3 = 15/6 = 5/2

Итого:
k = 13/14
b = 5/2

Шаг 4. Записываем уравнение прямой.

y = (13/14)x + 5/2

------------------------------------------------------------

Ответ

k = 13/14
b = 5/2

Прямая:
y = (13/14)x + 5/2

------------------------------------------------------------

Ответ на вопрос

«Минимизировать ошибку» означает взять вектор невязок:
r = (y1 - (k x1 + b), y2 - (k x2 + b), ..., yn - (k xn + b))

и сделать его длину (евклидову норму ||r||_2) как можно меньшей.

То есть мы выбираем прямую так, чтобы расстояние между вектором наблюдений y
и вектором предсказаний (kx + b) в пространстве R^n было минимальным в смысле L2-нормы.
```
